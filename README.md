# Skanda Plus Frontend

## Overview

Skanda Plus Frontend is a modern web application built with Next.js and React. It provides a responsive and user-friendly interface for an online learning platform where users can explore AI courses, corporate training programs, blogs, career opportunities, and contact information.

The application is designed with a focus on performance, scalability, and maintainability using a component-based architecture.

---

## Features

- Responsive and modern user interface
- AI course catalog
- Corporate training section
- Blog page
- Career opportunities page
- Contact page
- Reusable React components
- Custom 404 (Not Found) page
- Optimized image handling
- Clean project structure
- Fast navigation with Next.js App Router

## Installation

Clone the repository.

```bash
git clone https://github.com/your-username/your-repository.git
```

Navigate to the project directory.

```bash
cd frontend
```

Install dependencies.

```bash
npm install
```

---

## Running the Project

Start the development server.

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:4028
```

---

## Build for Production

```bash
npm run build
```

Start the production server.

```bash
npm run serve
```

## Component Architecture

The project follows a reusable component-based architecture.

Main reusable components include:

- Header
- Footer
- HeroSection
- CourseCard
- UI Components

This approach improves maintainability, consistency, and code reusability.

## Styling

The project uses Tailwind CSS for responsive design and utility-first styling. Global styles are managed using CSS files within the `src/styles` directory.
